# 中年人为什么难恋爱？
大家一定会说：因为责任重；压力大；造化弄人。
但结论很扎心——大部分中年人，都变丑了。
那天碰到一熟人。十年前他还是标致人儿，现在头顶比灯泡还亮，肚子比孕妇还大。
岁月这把杀猪刀，从来不会放过任何人。
年轻时谁不是满脸胶原蛋白？现在呢？女生的护肤品从保湿升级到抗衰，男生的发际线从额头退到头顶。照个镜子都要勇气，还指望别人对你一见钟情？
但这还不是最致命的。
最致命的是，我们变聪明了。
二十多岁谈恋爱，对方送朵花，都能感动三天。现在？他开宝马送你回家，你脑子里已经在估算他的车贷、年薪和孩子抚养费；她对你笑一下，你立刻分析这笑里有几分真心。
中年人摔过的跟头太多，知道风花雪月填不饱肚子，海誓山盟扫不清一地鸡毛。
一个朋友去相亲，回来吐槽：“他居然跟我聊区块链，我满脑子都在想他社保交够15年了吗。”
另一个更绝，约会对象请她吃日料，她一边吃一边算这顿够买多少奶粉。“我才意识到，我已经没办法单纯地享受一顿饭了。”
看，这就是中年人——我们太擅长解构爱情了。
爱情是什么？年轻时觉得是电光石火，现在知道是多巴胺分泌。心动是什么？年轻时觉得是缘分天定，现在明白是条件匹配。
我们像经验丰富的侦探，总能从任何暧昧中找出破绽。
他秒回信息？可能是身居闲职。她主动约饭？估计是饭票。他说“你很特别”？套路，都是套路。
不是我们变得cynic，是见过的套路太多，踩过的坑太深。
中年人心里都住着一个会计，时刻在计算投入产出比。
我们时间宝贵，精力有限。再也不会为了一句“我爱你”熬夜到天明，再也不会为了一个拥抱穿越半座城。
我们都太清楚，恋爱这笔投资，风险太高，回报太低。
更可怕的是，我们对自己也过于清醒了。
知道笑起来眼角有多少纹，知道沉默时有多憔悴，知道自己的坏脾气和怪癖，知道自己给不起的承诺和扛不起的责任。
一个清醒的丑人，是没法陷入感情的。
就像我那个离婚三年的同事说的：“不是不想谈恋爱，是再也找不到那种美得冒泡的傻冒状态了。”
是啊，我们活得太明白了，明白到连自己都骗不过。
或许偶尔会怀念吧。怀念因一句情话就失眠的夜晚，怀念敢爱敢恨的年纪。但怀念归怀念，真要重来？算了算了，明天还要早起开会。
于是，中年人的爱情，亡于又丑又精明。
而我们为此叹息的闲功夫都没有。
